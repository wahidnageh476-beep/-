# 🎓 تحليل قبول الجامعات المصرية

## 📌 نبذة عن المشروع
تحليل شامل لبيانات قبول 14 جامعة حكومية مصرية باستخدام Power BI و Python و SQL.
يهدف المشروع لمساعدة الطلاب على معرفة الجامعات المتاحة بناءً على مجموعهم.
